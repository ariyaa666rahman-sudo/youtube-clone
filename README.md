# youtube-clone# 🎥 YouTube Clone Project

A high-fidelity YouTube clone built as part of my Full-Stack Web Development journey. This project focuses on creating a responsive, modern UI using clean and semantic code.

## 🚀 Live Demo
You can view the live project here: [https://ariyaa666rahman-sudo.github.io/youtube-clone/](https://ariyaa666rahman-sudo.github.io/youtube-clone/)

## ✨ Features
* **Fully Responsive Design:** Works perfectly on Desktop, Tablet, and Mobile.
* **Modern UI/UX:** Replicated the YouTube aesthetic with precise spacing and layouts.
* **Semantic HTML:** Focused on best practices for SEO and accessibility.
* **Modular Structure:** Images and assets are organized in dedicated directories (`yt/thumbnails`).

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3
* **Tools:** VS Code, Git, GitHub Pages

## 📂 Project Structure
* `index.html` - The main entry point of the application.
* `yt/thumbnails/` - Contains all visual assets and icons.
